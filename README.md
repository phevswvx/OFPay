## 如何将OnlyFans界面设置为中文

虽然OnlyFans默认是英文界面，但你可以轻松地将其切换为中文。步骤如下：

1. 登录后，点击页面左下角的“More”选项。
2. 在语言设置中选择“Chinese Simplified”，页面会立即切换为中文。
